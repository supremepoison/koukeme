# TabBar 图标说明

请在当前目录下添加以下图标文件：

## 必需的图标文件 (尺寸: 81x81 px, PNG格式)

| 文件名 | 说明 |
|--------|------|
| `home.png` | 首页图标（未选中状态） |
| `home-active.png` | 首页图标（选中状态）|
| `chat.png` | 教练图标（未选中状态） |
| `chat-active.png` | 教练图标（选中状态）|
| `community.png` | 社区图标（未选中状态） |
| `community-active.png` | 社区图标（选中状态）|
| `profile.png` | 我的图标（未选中状态） |
| `profile-active.png` | 我的图标（选中状态）|

## 图标设计建议

- **风格**: 线性或扁平化设计
- **颜色**: 未选中用灰色(#999999)，选中用紫色(#667eea)
- **背景**: 透明背景
- **尺寸**: 81x81 像素 (建议导出为 @3x，实际绘制用 27x27 pt)

## 临时解决方案

如果没有图标文件，可以暂时删除 app.json 中的 tabBar 配置，小程序仍然可以正常运行，只是没有底部导航栏。
